Inject ToastService into component.
Toast types are :-
success, info, error, warning
Toast positions are :-
toast-top-right, toast-top-left, toast-bottom-right, toast-bottom-left, toast-top-center, toast-bottom-center, toast-top-full-width, toast-bottom-full-width
ActiveToast provides following options :-
{{col.label}}
{{data[col.value]}}
Sample code :-
showToaster() {
  this.toastr.success('Hello world!', 'Toastr fun!')
   .onTap
   .pipe(take(1))
   .subscribe(() => this.toasterClickedHandler());
}
toasterClickedHandler() {
  console.log('Toastr clicked');
}
